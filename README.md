# BracketClick Photo Booth

```bash
git clone <repo-url>
cd gdg

python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS/Linux

pip install -r requirements.txt
python app.py
```

Open **http://127.0.0.1:5000**. Ensure `hand_landmarker.task` is in the project folder.
